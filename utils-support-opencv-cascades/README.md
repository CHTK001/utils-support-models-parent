# utils-support-opencv-cascades

OpenCV Haar/LBP 级联分类器模型资源（人脸/眼睛/微笑等）

---

## 快速开始

### 1. 添加依赖

```xml
<dependency>
    <groupId>com.chua</groupId>
    <artifactId>utils-support-opencv-cascades</artifactId>
    <version>${project.version}</version>
</dependency>
```

---

## 配置说明

本模块为零配置模块，引入依赖后即可使用。
